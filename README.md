# Narrative Engine

A structured, literature-grounded framework for the generative production of narrative, powered by:
- **Greimasian actantial theory** — deep structural necessity (value-objects, canonical schema, action/state logic)
- **Proppian morphology** — functional sequence skeleton
- **Todorov equilibrium** — macro-arc validation
- **Complete narrative ontology** — 12 layers from brief to evaluation

## Core Principles

- **Theory-first architecture** — narrative is the product of deep structural constraints (fabula), not surface-level text generation.
- **Deterministic by design** — every story beat follows from logical necessity, not stochastic filler.
- **Agent-based orchestration** — a professional role stack of specialised agents (Showrunner, Structuralist, Character Architect, Character Simulator, Dialogue Specialist, etc.).
- **Two-gate evaluation** — hard gate (structural soundness) rejects broken narratives; soft gate (novelty, genre fit, thematic clarity) ranks sound candidates.
- **Layered character model** — function + personality (FFM) + values (Schwartz) + social mode (RMT) + attachment + motivation (SDT+Reiss) + emotion (Plutchik).
- **Action/State distinction (действие / състояние)** — only state-transforming operations are valid narrative units.
