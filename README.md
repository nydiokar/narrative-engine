# Narrative Engine

A structured, literature-grounded framework for the generative production of narrative, powered by Greimasian actantial theory, Proppian morphology, and strict causality-driven story logic.

## Core Principles

- **Theory-first architecture** — narrative is the product of deep structural constraints (fabula), not surface-level text generation.
- **Deterministic by design** — every story beat follows from logical necessity, not stochastic filler.
- **Agent-based orchestration** — a system of specialized agents (Director, Structuralist, Character Architect, etc.) collaborates to produce coherent, critique-refined narratives.

## Repository Structure

| Directory | Purpose |
|-----------|---------|
| `core/` | Foundational narrative models (Greimas, Propp, story-logic) |
| `contracts/` | Data contract schemas for all narrative entities |
| `agents/` | Agent role definitions and responsibilities |
| `workflows/` | End-to-end narrative generation pipelines |
| `templates/` | YAML templates for intake, bible, episodes, etc. |
| `stories/` | Generated and in-progress story artifacts |
| `research/` | Domain research and design rationale |
| `docs/` | System overview, theory notes, and reference |
| `archive/` | Deprecated or historical artifacts |
