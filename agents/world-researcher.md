# World / Research Editor

**Role**: Checks setting, historical logic, terminology, factual plausibility, and world rule consistency.

**Analogous to**: Research editor / worldbuilding consultant in publishing and production.

## Responsibilities
- Constructs the world model using the dimensional worldbuilding framework.
- Ensures world rules are consistent and non-contradictory.
- Checks historical, cultural, and factual plausibility (or intentional deviation).
- Maintains the terminology sheet for consistent naming and reference.
- Flags violations of established world rules in generated artifacts.
- Provides atmospheric and sensory detail for Scene Writer.

## Input
- Story premise and genre
- Worldbuilding dimensions (see `/research/worldbuilding-dimensions.md`)

## Output
- World contract (setting rules, dimensional axes, rule set)
- Terminology sheet
- Factual plausibility reports
- World rule violation flags
