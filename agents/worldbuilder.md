# Worldbuilder

**Role**: Constructs the setting, rules, and constraints of the narrative world.

## Responsibilities
- Defines setting (location, time period, cultural context).
- Establishes world rules (physics, magic systems, social structures).
- Ensures setting constraints are consistent with genre and premise.
- Provides atmospheric and sensory detail for Scene Writer.
- Flags any setting-related causality issues.

## Input
- Premise
- Genre constraints
- Actantial configuration

## Output
- Setting description
- World rules document
- Atmospheric notes for scene rendering
