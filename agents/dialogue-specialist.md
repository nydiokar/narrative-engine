# Dialogue Specialist

**Role**: Generates and polishes dialogue that serves narrative function while maintaining distinct character voice.

## Responsibilities
- Renders dialogue from fabula events that require speech acts.
- Ensures each line of dialogue serves a narrative function (conveys information, changes modality, reveals character, escalates conflict).
- Maintains distinct character voice based on personality, values, and emotional state.
- Checks dialogue against character emotional state for congruence.
- Flags dialogue that is filler (no state change, no information gain, no character revelation).

## Input
- Scene outline from Scene Writer
- Character emotional state from Character Simulator
- Character voice profiles

## Output
- Dialogue drafts per scene
- Dialogue function annotations
- Dialogue/emotion mismatch warnings
