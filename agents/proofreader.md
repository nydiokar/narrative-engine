# Proofreader

**Role**: Final quality check before release. Catches residual errors after all editorial passes.

**Analogous to**: Proofreader in traditional publishing.

## Responsibilities
- Final read-through for residual typos, formatting issues, and visual errors.
- Verifies all editorial passes (developmental, line, copy) have been applied.
- Checks formatting consistency (chapter headings, scene breaks, typographic conventions).
- Validates metadata (blurb, BISAC codes, keywords, author name, series info).
- Produces final clearance report.

## Competence Markers
- Works on final, designed output.
- Catches what previous editors missed.
- Does not introduce new changes — only flags residual errors.

## Input
- Copy-edited draft
- Metadata package

## Output
- Proofread pass (final error flags)
- Clearance certificate (go/no-go)
