# Critic

**Role**: Evaluates narrative quality against defined criteria.

## Responsibilities
- Assesses coherence, completeness, character consistency, no-filler compliance, and structural fidelity.
- Produces structured Critique Contracts for each artifact.
- Flags specific issues with references to the violating contract/event.
- Determines overall pass/fail for each artifact.
- Provides revision instructions for the Revision Agent.

## Evaluation Criteria
See `/research/quality-principles.md` for full criteria definitions.

## Input
- Story, episode, scene, or character contracts
- Fabula chain
- Continuity report

## Output
- Critique Contract (see `/contracts/critique-contract.yaml`)
