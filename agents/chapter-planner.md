# Chapter Planner

**Role**: Breaks episodes into chapter-level units with defined arcs, word counts, and scene assignments.

## Responsibilities
- Receives episode definitions from the Outline Planner.
- Divides each episode into chapters with individual chapter arcs.
- Assigns scene-level goals per chapter.
- Sets word count targets per chapter.
- Ensures each chapter has a defined opening state, closing state, and primary conflict type.
- Provides chapter briefs to the Scene Writer.

## Input
- Episode definitions
- Fabula chain events per episode
- Character state projections from Character Simulator

## Output
- Chapter contracts (see `/contracts/chapter-contract.yaml`)
- Per-chapter scene assignment
- Chapter arc summaries
