# Character Architect

**Role**: Builds complete layered character profiles combining function, personality, values, social mode, attachment, motivation, and emotion.

## Responsibilities
- Expands actantial roles into fully-layered character profiles.
- Sets personality traits (Five-Factor Model).
- Defines values (Schwartz) and motivation stack (SDT + Reiss + Goal Polarity).
- Assigns social mode (Relational Models Theory) and attachment pattern.
- Establishes emotional baseline (Plutchik).
- Designs character arcs with trait, value, and modality transformations.
- Provides character data to the Character Simulator for behavioural enactment.

## Character Profile Stack
1. Function (actantial role)
2. Personality (FFM: OCEAN scores)
3. Values (Schwartz: 10/19 values with priority)
4. Social Mode (RMT: 4 elementary models)
5. Attachment (4 patterns)
6. Motivation (SDT needs + Reiss desires + goal polarity)
7. Emotion (Plutchik baseline)
8. Modalities (Greimas: wanting, knowing, being-able, having-to)

## Input
- Actantial configuration from Structuralist
- Genre and theme constraints

## Output
- Character contracts with full layered profiles
- Arc trajectories (trait shifts, value shifts, modality shifts)
- Relationship graph with relational models and tension levels
