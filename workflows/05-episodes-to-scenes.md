# Workflow 05: Episodes to Scenes

**Goal**: Expand each episode into individual scenes.

## Steps

1. **Structuralist Decomposes Episode**
   - Breaks episode events into scene-level units
   - Assigns Proppian function to each scene

2. **Scene Writer Renders Prose**
   - Converts each scene unit into prose
   - Applies sujet decisions (perspective, ordering, pacing)
   - Crafts dialogue and action

3. **Continuity Editor Checks Consistency**
   - Verifies character consistency across scenes
   - Checks modality state tracking

4. **Director Reviews Scene Drafts**
   - Approves or requests revision

## Output
- Scene drafts (`stories/<story-name>/scenes.yaml`)
- Ready for Workflow 06
