# Workflow 03: Premise to Structure

**Goal**: Derive deep narrative structure (fabula) from the approved premise.

## Steps

1. **Structuralist Builds Fabula**
   - Selects Proppian function sequence
   - Defines narrative programs
   - Establishes causality chains
   - Applies modality rules

2. **Structuralist Checks Constraints**
   - Validates against story-logic rules (causality, modality, no-filler)
   - Ensures genre compliance

3. **Director Reviews Structure**
   - Validates fabula coherence
   - Approves or requests revision

## Output
- Fabula chain (`stories/<story-name>/structure.yaml`)
- Ready for Workflow 04
