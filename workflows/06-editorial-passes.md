# Workflow 06: Editorial Passes

**Goal**: Apply professional editorial passes in sequence — developmental, line, copy, proof.

## Steps

### 6a. Developmental Edit
1. **Developmental Editor Evaluates Draft**
   - Assesses narrative structure, pacing, act architecture
   - Checks genre expectations and story promise delivery
   - Produces structural revision recommendations

2. **Revision Agent Applies Structural Changes**
   - Implements developmental edit recommendations
   - Updates contracts as needed

3. **Re-critique** (if structural changes were significant)

### 6b. Line Edit
1. **Line Editor Refines Prose**
   - Improves sentence rhythm, diction, metaphor density
   - Preserves authorial voice
   - Flags weak constructions

2. **Revision Agent Applies Line Changes**

### 6c. Copy Edit
1. **Copy Editor Checks Consistency**
   - Grammar, spelling, punctuation
   - Cross-references, timeline, names, terminology
   - Fictional world consistency
   - Style guide adherence

2. **Revision Agent Applies Copy Changes**

### 6d. Proofread
1. **Proofreader Final Check**
   - Residual typos and formatting issues
   - Chapter headings, scene breaks, typographic conventions
   - Metadata validation

2. **Clearance Certificate**

## Output
- Editorially polished draft
- Editorial pass reports (developmental, line, copy, proof)
- Ready for Workflow 07 (Critique and Evaluation)
