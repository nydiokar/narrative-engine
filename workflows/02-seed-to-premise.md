# Workflow 02: Seed to Premise

**Goal**: Transform raw intake into a structured premise with actantial configuration.

## Steps

1. **Director Reviews Intake**
   - Assesses premise viability
   - Assigns genre constraints

2. **Structuralist Analyzes Premise**
   - Identifies actantial configuration
   - Maps initial actant roles

3. **Character Architect Outlines Protagonists**
   - Drafts initial character sketches
   - Sets preliminary modality states

4. **Director Approves Premise**
   - Confirms actantial configuration
   - Signs off on premise viability

## Output
- Approved actantial configuration
- Initial character sketches
- Ready for Workflow 03
