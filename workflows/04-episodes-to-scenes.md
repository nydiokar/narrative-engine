# Workflow 04: Episodes to Scenes

**Goal**: Simulate, plan, and write scenes with full character and dialogue support.

## Steps

1. **Character Simulator Enacts Episode**
   - Loads character profiles for the episode
   - Simulates character reactions to each event
   - Generates character-state vectors per scene
   - Flags character-inconsistent developments

2. **Dialogue Specialist Plans Speech Acts**
   - Identifies scenes requiring dialogue
   - Generates dialogue function annotations
   - Ensures each speech act serves narrative purpose

3. **Scene Writer Renders Prose**
   - Converts each scene unit into prose
   - Incorporates character state and emotional data
   - Applies discourse contract (POV, focalisation, tense)
   - Crafts dialogue and action

4. **Scene Writer Runs Greimas Diagnostic**
   - Answers 5 questions per scene
   - Confirms state transformation
   - Self-assesses diagnostic pass/fail

5. **Continuity Editor Checks Consistency**
   - Verifies character, causality, world rule, and discourse consistency
   - Runs hard gate checks

## Output
- Scene drafts with Greimas diagnostics
- Character state vectors per scene
- Dialogue with function annotations
- Continuity report
- Ready for Workflow 05
