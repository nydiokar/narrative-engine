# Workflow 04: Structure to Episodes

**Goal**: Decompose the fabula into episodic segments.

## Steps

1. **Structuralist Segments Fabula**
   - Groups events into episodes
   - Assigns narrative programs to episodes
   - Ensures each episode has a clear function in the overall arc

2. **Character Architect Refines Arcs**
   - Maps character arc milestones to episodes
   - Defines modality shift per episode

3. **Worldbuilder Provides Setting Per Episode**
   - Assigns location and atmosphere to each episode
   - Ensures setting supports narrative program

4. **Director Approves Episode Structure**

## Output
- Episode definitions (`stories/<story-name>/episodes.yaml`)
- Ready for Workflow 05
