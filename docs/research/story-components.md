# Story Components

A narrative is composed of irreducible components, ordered by the Greimas-Propp integration. The types_maps.md adds plot structure systems as available macro-grammars.

## Greimas Layer — Deep Structural Necessity

1. **Actants** — the deep roles that drive action (Subject, Object, Sender, Receiver, Helper, Opponent). Actants are structural positions, not fixed characters.
2. **Objects of Value** — value-states under contest (recognition, freedom, truth, revenge, belonging, etc.). Not physical items — abstract qualities that actants desire, defend, block, or redefine.
3. **Modalities** — the qualifications of action: wanting (vouloir), knowing (savoir), being-able (pouvoir), having-to (devoir).
4. **Canonical Schema Phases** — every narrative program cycles through Manipulation → Competence → Performance → Sanction.
5. **Action / State (действие / състояние)** — the fundamental unit: actions transform states. No "events" — only state-transforming operations.

## Propp Layer — Functional Sequence Morphology

6. **Proppian Functions** — a vocabulary of 32 canonical story-beat types that provide one possible morphology for how the Greimas-defined movement unfolds.

## Alternative Plot Structure Systems

The engine supports multiple structural grammars. Propp is the default for folktale/quest/transformation structures, but these alternatives are available:

| System | Best For | Key Concept |
|--------|----------|-------------|
| **Propp Morphology** | Folktale, quest, transformation | 32 functions in stable order |
| **Greimas Actantial Model** | Any narrative with clear value-contest | 6 actant roles |
| **Greimas Canonical Schema** | Any narrative with formal arc | 4 phases: Manipulation → Competence → Performance → Sanction |
| **Todorov Equilibrium** | Simple macro-arc validation | Equilibrium → Disruption → Restoration |
| **Three-Act Structure** | Screenwriting, commercial fiction | Setup → Confrontation → Resolution |
| **Five-Act Structure** | Shakespearean, epic | Exposition → Rising → Climax → Falling → Denouement |
| **Hero's Journey** | Mythic, fantasy, adventure | 12 stages from Ordinary World to Return |
| **Booker's Seven Plots** | Meta-classification | Overcoming the Monster, Rags to Riches, Quest, Voyage and Return, Comedy, Tragedy, Rebirth |
| **Polti's 36 Dramatic Situations** | Scene-level dramatic kernel | Supplication, Deliverance, Vengeance, etc. |
| **Save the Cat Beat Sheet** | Screenwriting, commercial fiction | 15 beats from Opening Image to Final Image |
| **Kishōtenketsu** | East Asian narrative structure | Introduction → Development → Twist → Reconciliation |
| **Fichtean Curve** | Literary fiction | Rising tension with mini-crises |
| **Dan Harmon Story Circle** | Television, iterative structure | 8-step circle from Comfort to Change |
| **Genre-Specific** | Mystery, thriller, romance, bildungsroman | Investigation structure, escalation, union/separation, maturation |

## Surface Layer — Rendering

7. **Temporal Relations** — ordering, duration, frequency of events (sujet level).
8. **Causal Relations** — the logical connective tissue between events, verified by both Greimasian necessity and Proppian sequence.

These components are assembled into a fabula (validated by Greimas, sequenced by a chosen morphology), which is then rendered as a sujet.
