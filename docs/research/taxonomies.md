# Taxonomies

## Actantial Role Taxonomy

- **Subject**: Hero, Protagonist, Seeker
- **Object**: Goal, Treasure, Loved One, Knowledge, Power
- **Sender**: King, Mentor, Destiny, Society
- **Receiver**: The Subject, Society, A Loved One
- **Helper**: Ally, Magic Artifact, Knowledge, Skill
- **Opponent**: Villain, Obstacle, Internal Flaw, Nature

## Proppian Function Categories

- **Preparation** (α–θ): introduction, interdiction, violation
- **Complication** (A–C): villainy, lack, mediation
- **Transference** (D–G): testing, reaction, receipt of agent
- **Struggle** (H–I): combat, branding
- **Return** (J–K): victory, resolution of lack
- **Reception** (L–N): return, pursuit, rescue
- **Recognition** (O–T): unrecognized arrival, claim, task, solution
- **Exposure** (U–W): exposure, transfiguration, punishment, wedding
