# Stakes Types

Stakes define what is gained or lost. Every scene must have active stakes — a value-object under contest. The stakes type determines the intensity and nature of the conflict.

## Stakes Types

| Stake | Polarity | Domain | Typical Conflict Type |
|-------|----------|--------|-----------------------|
| **Life / Death** | Survival vs Extinction | Physical | Environmental, Interpersonal |
| **Freedom / Captivity** | Autonomy vs Control | Social/Political | Institutional, Systemic |
| **Love / Loss** | Union vs Separation | Emotional/Relational | Interpersonal, Internal |
| **Truth / Ignorance** | Knowledge vs Deception | Epistemic | Epistemic, Interpersonal |
| **Honor / Shame** | Dignity vs Disgrace | Social/Moral | Interpersonal, Institutional |
| **Belonging / Exile** | Inclusion vs Rejection | Social/Emotional | Interpersonal, Internal |
| **Power / Subjugation** | Control vs Powerlessness | Political/Personal | Institutional, Systemic |
| **Identity / Erasure** | Selfhood vs Annihilation | Existential | Internal, Metaphysical |
| **Memory / Oblivion** | Continuity vs Forgetting | Existential/Epistemic | Metaphysical, Internal |
| **Order / Chaos** | Stability vs Anarchy | Social/Cosmic | Systemic, Environmental |
| **Justice / Corruption** | Fairness vs Injustice | Moral/Political | Institutional, Interpersonal |
| **Survival / Extinction** | Continuation vs Annihilation | Physical/Existential | Environmental, Systemic |
| **Family / Separation** | Kinship vs Isolation | Relational | Interpersonal, Internal |
| **Community / Collapse** | Cohesion vs Disintegration | Social | Systemic, Institutional |
| **Soul / Damnation** | Redemption vs Condemnation | Moral/Spiritual | Internal, Metaphysical |
| **Legacy / Meaninglessness** | Significance vs Futility | Existential | Internal, Metaphysical |
| **Future / Stagnation** | Progress vs Paralysis | Temporal/Social | Systemic, Internal |

| **Recognition / Invisibility** | Acknowledgment vs Ignorance | Social/Emotional | Interpersonal, Internal |
| **Autonomy / Control** | Self-direction vs Domination | Personal/Political | Institutional, Internal |

## Stakes Selection Rules

- Every scene must have at least one active stake type.
- The stake type should derive from the episode's Object of Value.
- Stakes can escalate across a sequence (e.g., Love/Loss → Life/Death).
- Conflicting stakes between actants (one wants Belonging, the other wants Power) generate strong conflict.
- The Critic checks that stakes are present and appropriately scaled to the scene type.
