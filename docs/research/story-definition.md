# Definition of a Strong Story

This document consolidates the engine's quality criteria into a single operational definition.

## A Strong Story Has

| Quality | Check |
|---------|-------|
| **Clear Object of Value** | The reader knows what is being contested and why it matters. |
| **Specific Protagonist Desire** | The Subject wants something concrete, not vague. |
| **Pressure from Opposition** | The Opponent has their own value-logic and blocks actively. |
| **Escalating Cost** | Each victory or defeat changes the stakes. |
| **Meaningful State Changes** | Every scene transforms at least one state parameter. |
| **Causal Episode Chain** | No episode could be removed without breaking the fabula. |
| **World That Pressures Theme** | The setting's rules force thematic choices. |
| **Characters with Contradiction** | No character is purely consistent — internal tension drives arcs. |
| **Scenes with Turns** | Every scene shifts the actantial configuration or modality state. |
| **Ending That Sanctions the Journey** | The final junction is a meaningful consequence of the Performance. |
| **Emotional Payoff** | The reader feels the weight of the transformation. |
| **Reader-Contract Fulfillment** | The genre promise is delivered (or deliberately subverted with purpose). |
| **Freshness Without Randomness** | Novelty arises from recombination and cost, not arbitrary twists. |

## A Weak Story Has

| Failure | Symptom |
|---------|---------|
| **Events Without Transformation** | Scenes occur but no state changes. |
| **Conflict Without Value** | Characters fight but nothing is actually contested. |
| **Villain Without Logic** | The Opponent obstructs without their own value-relationship to the Object. |
| **World Without Pressure** | The setting is decorative — it does not force hard choices. |
| **Theme Without Dramatization** | The theme is stated but never embodied in action or choice. |
| **Characters Without Agency** | Events happen to the Subject; they do not make consequential decisions. |
| **Scenes Without Turns** | The scene ends in the same actantial configuration it began. |
| **Plot Solved by Coincidence** | The resolution depends on luck, not character action. |
| **Ending Without Cost** | Victory is free; nothing was risked. |
| **Originality Through Randomness** | Novelty comes from arbitrary deviation, not meaningful recombination. |
| **Cliché Through Default Choices** | Every selection is the genre default, unexamined. |

## The Greimas-Gate Summary

A story is strong if, for every episode and scene:

1. **A Subject desires an Object of value.**
2. **An Opponent with their own value-logic blocks access.**
3. **State changes occur through action (действие → състояние).**
4. **The canonical schema completes (Manipulation → Competence → Performance → Sanction).**
5. **Costs are paid.**
6. **The ending is a meaningful sanction of the Subject's choices.**
