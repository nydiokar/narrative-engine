# Relationship Archetypes

Relationships between characters are defined by their archetypal pattern, which determines the relational model (RMT), the dominant conflict type, and the expected tension trajectory.

## Relationship Archetypes

| Archetype | Relational Model | Typical Tension Driver | Common Arc |
|-----------|-----------------|------------------------|------------|
| **Mentor / Student** | Authority Ranking | Knowledge transfer, independence | Student surpasses or rejects mentor |
| **Parent / Child** | Authority Ranking / Communal Sharing | Autonomy vs protection, inheritance | Child individuates or inherits burden |
| **Siblings** | Equality Matching / Communal Sharing | Rivalry, comparison, shared history | Rivalry → respect or rupture |
| **Rivals** | Equality Matching | Competition for same Object | Respect or destruction |
| **Enemies** | Market Pricing (negative) | Incompatible value-logics | Elimination or understanding |
| **Lovers** | Communal Sharing | Intimacy vs autonomy, jealousy | Union, separation, or transformation |
| **Forbidden Lovers** | Communal Sharing + External Opposition | Social order vs personal desire | Sacrifice, rebellion, or tragedy |
| **Protector / Protected** | Authority Ranking | Safety vs freedom, dependence | Dependency → equality or breakdown |
| **Captor / Captive** | Authority Ranking (coerced) | Power, survival, resistance | Stockholm, escape, or overthrow |
| **Master / Servant** | Authority Ranking | Loyalty vs conscience, class | Rebellion or mutual recognition |
| **Leader / Follower** | Authority Ranking | Trust, direction, betrayal | Disillusionment or vindication |
| **Commander / Soldier** | Authority Ranking | Orders vs morality, sacrifice | Obedience with cost or mutiny |
| **Creator / Creation** | Authority Ranking | Responsibility, control, abandonment | Creation rebels or fulfills |
| **Judge / Accused** | Authority Ranking | Guilt, truth, justice | Condemnation or exoneration |
| **Healer / Wounded** | Communal Sharing + Authority Ranking | Dependency, recovery, boundaries | Healing or codependence |
| **Tempter / Tempted** | Market Pricing | Desire, integrity, cost | Fall or resistance |
| **Believer / Skeptic** | Equality Matching | Faith vs evidence, certainty vs doubt | Conversion or confirmation |
| **Traitor / Betrayed** | Market Pricing (broken trust) | Trust, revenge, forgiveness | Rupture, revenge, or reconciliation |
| **Double / Mirror** | Equality Matching (reflexive) | Identity, self-knowledge, confrontation | Integration or destruction |
| **Hunter / Hunted** | Market Pricing (predatory) | Survival, fear, power | Escape, reversal, or destruction |
| **Old Self / New Self** | Internal (intrapersonal) | Identity continuity vs transformation | Integration or rejection of past |
| **Civilized / Wild** | Equality Matching (opposing natures) | Control vs freedom, order vs nature | Reconciliation or conquest |
| **Insider / Outsider** | Communal Sharing (exclusion) | Belonging, identity, loyalty | Assimilation, rejection, or transformation |

## Relationship Arc Types

| Arc | Trajectory | Typical Archetypes |
|-----|-----------|-------------------|
| **Strangers → Allies** | Trust built through shared trial | Any |
| **Allies → Enemies** | Trust broken by value divergence | Friends, siblings, mentor/student |
| **Enemies → Lovers** | Hostility recontextualised as attraction | Rivals, enemies, hunter/hunted |
| **Rivals → Mutual Respect** | Competition reveals equal capability | Rivals, siblings |
| **Mentor → Betrayed Authority** | Student exceeds or rejects mentor | Mentor/student |
| **Protector → Controller** | Protection curdles into domination | Protector/protected, parent/child |
| **Dependent → Independent** | Protected character claims autonomy | Protector/protected, parent/child |
| **Idealization → Disillusionment** | Admired figure is revealed as flawed | Mentor/student, leader/follower, lovers |
| **Betrayal → Forgiveness** | Trust broken then rebuilt | Any |
| **Love → Sacrifice** | Love requires giving up the beloved | Lovers, forbidden lovers, parent/child |
| **Friendship → Ideological Rupture** | Values diverge beyond repair | Allies, siblings, believers/skeptics |
| **Kinship → Chosen Separation** | Blood ties severed by necessity | Siblings, parent/child |
| **Captivity → Understanding** | Captor and captive see each other's humanity | Captor/captive |
| **Mutual Use → Genuine Care** | Transactional relationship becomes emotional | Master/servant, tempter/tempted |
