# Symbolism and Motifs

## Symbolic Functions

| Function | Description |
|----------|-------------|
| **Identity Marker** | An object that signifies who a character is |
| **Buried Truth** | An object that conceals or reveals a hidden fact |
| **Inheritance** | An object that carries a legacy or burden |
| **Temptation** | An object that offers desire at a cost |
| **Authority** | An object that represents power or legitimacy |
| **Memory** | An object that holds or triggers remembrance |
| **Guilt** | An object that embodies a character's transgression |
| **Transformation** | An object whose change mirrors a character's arc |
| **Protection** | An object that keeps a character safe |
| **Corruption** | An object that morally compromises its bearer |
| **Recognition** | An object that proves identity or worth |
| **Passage** | An object that grants access to a place or state |
| **Boundary** | An object that marks a limit or threshold |
| **Loss** | An object whose absence defines a character |
| **Return** | An object that reappears to close a narrative loop |
| **False Promise** | An object that appears valuable but is hollow |
| **Hidden Wound** | An object that symbolises an unhealed trauma |

## Motif Usage Rules

- A motif should appear at least three times across the narrative (establish → develop → pay off).
- The motif's meaning should shift as the fabula progresses (e.g., a key that first means "locked out" later means "chosen").
- Motifs must be grounded in the world — not imposed as arbitrary symbolism.
- The Critic checks that motifs have narrative function, not decorative presence.
