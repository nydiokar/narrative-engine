# Scene Types

Every scene serves a specific narrative function. The scene type determines the actantial configuration, the type of state transformation expected, and the likely emotional tone.

## Scene Type Catalogue

| Type | Function | Expected Transformation | Typical Canonical Phase |
|------|----------|------------------------|------------------------|
| **Introduction** | Establishes baseline state, characters, setting | State of order / equilibrium defined | Manipulation |
| **Inciting Scene** | Disrupts the equilibrium; introduces the Object of Value | Lack or desire introduced | Manipulation |
| **Decision Scene** | Subject commits to pursuing the Object | Modality of wanting established | Manipulation |
| **Discovery Scene** | Subject gains new information about the Object or world | Modality of knowing gained | Competence |
| **Confrontation Scene** | Subject engages Opponent directly | Value-object contested; modalities tested | Performance |
| **Interrogation Scene** | Information is extracted or withheld | Epistemic state changes | Competence / Performance |
| **Seduction Scene** | Subject is tempted toward or away from the Object | Modality shift (wanting corrupted or reinforced) | Competence |
| **Betrayal Scene** | An actant shifts position; trust is broken | Actantial reconfiguration | Performance |
| **Reversal Scene** | The situation inverts; Subject becomes Opponent or vice versa | Actantial position swap | Performance |
| **Revelation Scene** | A hidden truth about the Object of Value is exposed | Object of Value redefined | Sanction |
| **Training Scene** | Subject acquires a modality (knowing, being-able) | Modality gained | Competence |
| **Test Scene** | Subject's competence is proved or challenged | Modality validated or shown insufficient | Competence / Performance |
| **Negotiation Scene** | Actants bargain over the Object of Value | Junction terms redefined | Performance |
| **Escape Scene** | Subject flees a threatening force | Spatial/temporal state change | Performance |
| **Chase Scene** | Subject pursues or is pursued | Spatial relation changes; stakes escalate | Performance |
| **Battle Scene** | Direct physical or ideological clash | Victory/defeat junction change | Performance |
| **Quiet Aftermath** | Reflection on the consequences of action | Emotional/sanction state change | Sanction |
| **Recognition Scene** | Subject's true nature or achievement is acknowledged | Identity/status junction change | Sanction |
| **Judgment Scene** | An actant's actions are evaluated | Sanction rendered | Sanction |
| **Sacrifice Scene** | Subject gives up the Object or something of value | Junction disjunction; value redefined | Performance / Sanction |
| **Temptation Scene** | Subject is offered a false or costly Object | Modality of wanting tested | Competence |
| **False Victory Scene** | Subject appears to have achieved the Object | Apparent conjunction (later inverted) | Performance |
| **Dark Night Scene** | Subject experiences lowest point; all seems lost | Maximum disjunction; modality crisis | Performance |
| **Climax Scene** | The central conflict is resolved | Final junction change | Performance |
| **Resolution Scene** | New equilibrium is established | State of order restored or redefined | Sanction |
| **Epilogue Scene** | Final perspective on the transformed world | Final emotional/cognitive state | Sanction |

## Scene Selection Rules

- A scene type must serve the current canonical phase of its narrative program.
- No two consecutive scenes may share the same scene type unless separated by a different type.
- The scene type determines which Greimas diagnostic question set applies.
- Scene types have default emotional tone mappings (e.g., Dark Night → sadness/grief, Battle → anger/fear).
