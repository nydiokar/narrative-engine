2.3 Greimas actants
Subject = pursues Object
Object = value sought
Sender = initiates quest / gives mandate
Receiver = benefits from Object
Helper = enables Subject
Opponent = blocks Subject

Important:

Character ≠ actant.
One character can shift actantial position.
One actant can be distributed across multiple characters, institutions, objects, or internal forces.

Greimas is especially useful because the Object can be abstract: truth, legitimacy, love, recognition, freedom, power, restoration.

4.2 Greimas conflict grammar
Subject desires Object.
Opponent blocks access to Object.
Helper increases Subject’s modality.
Sender legitimizes pursuit.
Receiver benefits from success.
Counter-subject desires same Object or opposite value.

Conflict becomes precise when defined as:

Actant A wants value-state X.
Actant B prevents, corrupts, hides, redirects, monopolizes, or redefines X.

11. Matrix compatibility rules
11.1 Core matrix
Seed → Object of Value → Theme → Genre → World → Character wound → Conflict → Ending

Every story should be checked through this chain.

Example:

Seed: A child discovers the city is built on erased memories.
Object: truth / memory
Theme: truth vs comfort
Genre: mystery fantasy / dystopia
World: controlled city
Wound: inherited ignorance
Conflict: person vs institution
Ending: revelation with social cost
11.2 Good-fit patterns
Revenge + shame wound + tragedy = strong
Revenge + comic tone + no moral cost = weak unless satire

Freedom theme + prison world + control antagonist = strong
Freedom theme + open world + passive protagonist = weak

Love theme + forbidden relationship + social order = strong
Love theme + no obstacle = weak

Truth theme + mystery structure + deceptive world = strong
Truth theme + transparent world = weak

Power theme + ambitious protagonist + corrupting object = strong
Power theme + passive protagonist = weak
11.3 Productive clashes
Soft character in brutal world
Comic tone over tragic structure
Sacred theme in profane world
Romance inside political collapse
Innocent protagonist in corrupt system
Mystery inside fantasy world
Horror inside domestic realism
Epic stakes through intimate POV
11.4 Destructive clashes
Theme says freedom, plot rewards obedience without irony
Protagonist wants nothing, but story demands quest
World rules forbid solution, but ending ignores rules
Villain blocks nothing specific
Scenes happen but no state changes
Tone promises realism, plot depends on coincidence
Romance promised, intimacy never tested
Mystery promised, clues are withheld unfairly

12.2 Freshness generators
Change Object of Value
Change moral cost
Change actant roles
Invert helper/opponent
Make world enforce theme
Give antagonist a valid value
Move genre promise into unexpected world
Make victory damage something real
Make wound contradict desire
Make helper dangerous
Make love interest ideologically opposed
Make final sanction ambiguous
Make protagonist right tactically but wrong morally
Make antagonist wrong methodologically but right diagnostically
12.3 Freshness formula
Fresh = familiar structure + unexpected value logic + specific cost

Bad freshness:

Random twist
Arbitrary subversion
Genre betrayal
Shock without consequence

Good freshness:

A twist that reveals the true Object of Value.
A reversal that changes actantial positions.
A victory that exposes the protagonist’s false belief.

13.3 Selection rules
Choose Object first if the seed is symbolic.
Choose conflict first if the seed is action-based.
Choose world first if the seed is setting-based.
Choose character wound first if the seed is psychological.
Choose genre first only if the seed is market/audience-based.
13.4 Rejection rules

Reject an option if:

It does not pressure the protagonist.
It does not clarify the Object of Value.
It does not create scene conflict.
It contradicts genre promise without purpose.
It makes the ending obvious too early.
It solves the story too easily.
It adds lore without action-state consequences.

18. Definition of a good story/book

A strong story has:

Clear Object of Value
Specific protagonist desire
Pressure from opposition
Escalating cost
Meaningful state changes
Causal episode chain
World that pressures theme
Characters with contradiction
Scenes with turns
Ending that sanctions the journey
Emotional payoff
Reader-contract fulfillment
Freshness without randomness

A weak story has:

Events without transformation
Conflict without value
Villain without logic
World without pressure
Theme without dramatization
Characters without agency
Scenes without turns
Plot solved by coincidence
Ending without cost
Originality through randomness
Cliché through default choices
19. Final master prompt structure placeholder
You are operating the Book-Building OS.

Use these layers in order:

1. Parse human intake.
2. Identify Object of Value.
3. Generate genre/theme/world candidates.
4. Select best-fit combination using compatibility rules.
5. Build Greimas actantial model.
6. Build narrative programs and counter-programs.
7. Build Propp/plot morphology where useful.
8. Build character psychology and arcs.
9. Build world rules that pressure the theme.
10. Build episode chain.
11. Validate every episode for action-state transformation.
12. Build scene chain.
13. Draft.
14. Critique.
15. Revise.

Never create scenes that do not transform state.
Never create conflict without Object of Value.
Never create characters without desire, fear, wound, and function.
Never add lore unless it changes action possibilities.
Never pursue freshness through randomness.
