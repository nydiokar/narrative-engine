# Glossary

| Term | Definition |
|------|------------|
| **Actant** | A deep structural role in the narrative (e.g., Subject, Object, Opponent). |
| **Actantial Model** | Greimas's six-role schema for narrative structure. |
| **Fabula** | The raw, chronologically ordered sequence of events in a narrative. |
| **Sujet** | The presented order and stylistic rendering of fabula events. |
| **Narrative Program (NP)** | A formal transformation from an initial state to a terminal state. |
| **Proppian Function** | One of 31 canonical narrative actions identified by Vladimir Propp. |
| **Actantial Junction** | The state of connection (conjunction/disjunction) between an actant and an Object of Value. |
| **Modality** | The mode of being (wanting, knowing, being-able, having-to) that qualifies an actant's state. |
| **No-Filler Principle** | The constraint that every scene must advance the narrative; no disposable content. |
| **Critique Contract** | A structured report produced by the Critic agent assessing narrative quality. |
