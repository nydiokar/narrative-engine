# Glossary

| Term | Definition |
|------|------------|
| **Actant** | A deep structural role in the narrative (Subject, Object, Sender, Receiver, Helper, Opponent). Not a fixed "character" — actantial positions can shift. |
| **Actantial Model** | Greimas's six-role schema for narrative structure. |
| **Object of Value** | The value-state under contest — love, recognition, freedom, truth, revenge, belonging, etc. |
| **Value-State** | An abstract quality or condition that actants desire, defend, block, or redefine. |
| **Canonical Narrative Schema** | Manipulation/Contract → Competence/Qualification → Performance → Sanction. |
| **действие (Action)** | An operation that transforms a narrative state. |
| **състояние (State)** | A configuration of actantial junctions, modality values, and Object-of-value relations. |
| **Fabula** | The raw, chronologically ordered sequence of state-transforming operations. |
| **Sujet** | The presented order and stylistic rendering of fabula events. |
| **Discourse Layer** | POV, focalisation, tense, temporal handling, exposition strategy, suspense strategy, dialogue mode, voice. |
| **Narrative Program (NP)** | A formal transformation following the canonical schema. |
| **Proppian Function** | One of 32 canonical narrative actions. Used as sequence morphology over Greimas. |
| **Todorov Equilibrium** | Macro-structural pattern: equilibrium → disruption → restored equilibrium. |
| **Five-Factor Model (FFM)** | Big Five personality traits: Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism. |
| **Schwartz Values** | Ten broad values (with 19-value refinement) that drive character value-logic. |
| **Reiss 16 Desires** | Sixteen basic human desires with intensity scoring for character motivation. |
| **Self-Determination Theory (SDT)** | Three basic psychological needs: autonomy, competence, relatedness. |
| **Relational Models Theory (RMT)** | Four elementary social models: Communal Sharing, Authority Ranking, Equality Matching, Market Pricing. |
| **Plutchik's Wheel** | Eight basic emotions with intensity levels and combinatorial blends. |
| **Conflict Types** | Internal, Interpersonal, Institutional, Environmental, Epistemic, Metaphysical, Systemic. |
| **BISAC** | Book Industry Standards and Communications — industry genre taxonomy. |
| **Literary Theme Ontology (LTO)** | ~2,990 defined literary themes in a searchable hierarchy. |
| **GOLEM** | Narrative ontology modelling characters, relationships, events, settings, and inference. |
| **Hard Gate** | Structural soundness checks (causality, intentionality, consistency, stakes). |
| **Soft Gate** | Quality ranking (genre fit, thematic clarity, novelty, emotional transport). |
| **Cliché** | High-frequency genre defaults without inversion, escalation, recombination, or thematic necessity. |
| **Showrunner** | Top-level AI controller — owns creative voice, canon, tone, and final decisions. |
| **Script Editor** | Development editor — supports writer, preserves voice, ensures continuity. |
| **Developmental Editing** | Big-picture work on structure, content, organisation, genre. |
| **Line Editing** | Sentence/paragraph level language and style improvement. |
| **Copy Editing** | Grammar, style, cross-references, timeline, fictional world consistency. |
| **Proofreading** | Final quality check on designed pages or proofs. |
| **No-Filler Principle** | Every scene must transform a state or be causally necessary for a future transformation. |
| **Critique Contract** | Structured evaluation report with Greimas Fabula Coherence Engine diagnostics. |
