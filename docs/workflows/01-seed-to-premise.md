# Workflow 01: Seed to Premise

**Goal**: Transform the brief into a structured premise with actantial configuration and backbone choice.

## Steps

1. **Showrunner Reviews Brief**
   - Assesses premise viability
   - Assigns genre and theme constraints

2. **Structuralist Analyzes Premise**
   - Identifies actantial configuration
   - Maps initial actant roles to value-objects
   - Applies Todorov equilibrium model

3. **Backbone Choice**
   - Select structural backbone (Propp or alternative)
   - Define macro-structural grammar

4. **Character Architect Outlines Protagonists**
   - Drafts initial layered character profiles
   - Sets preliminary modality states
   - Assigns FFM traits and Schwartz values

5. **Showrunner Approves Premise**
   - Confirms actantial configuration
   - Signs off on premise viability

## Output
- Approved actantial configuration
- Backbone grammar selection
- Initial layered character sketches
- Ready for Workflow 02
