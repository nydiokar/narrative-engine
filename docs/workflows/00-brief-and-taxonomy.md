# Workflow 00: Brief and Taxonomy Fill

**Goal**: Capture the full narrative brief and select from available taxonomies.

## Steps

1. **Operator Provides Seed Brief**
   - Medium (novel)
   - Audience band (MG/YA/adult)
   - Language and style baseline
   - Hard content limits
   - Seed premise or logline
   - Desired emotional or thematic core
   - Preferred realism/fantasy/science range
   - Target length band
   - Standalone or series
   - Tolerance for darkness, sexual content, violence, ending mode
   - Inviolable world, setting, or character constraints

2. **Theme Specialist Selects Themes**
   - Query Literary Theme Ontology
   - Select primary (1-3) and secondary (0-5) themes
   - Define moral tensions between conflicting themes

3. **Theme Specialist Selects Genre**
   - Choose primary and secondary BISAC codes
   - Validate genre promise consistency

4. **World Researcher Sets World Axes**
   - Define worldbuilding dimension positions
   - Establish world rules and constraints

5. **Character Architect Prepares Layer Settings**
   - Select personality model ranges (FFM)
   - Configure motivation stack defaults
   - Set emotional palette baseline

6. **Showrunner Approves Brief**
   - Validates coherence of all choices
   - Signs off on concept layer

## Output
- Completed story contract (brief layer + concept layer)
- Theme contract
- Genre selection (BISAC)
- World axes
- Character layer configuration
