# Workflow 02: Premise to Structure

**Goal**: Derive deep narrative structure (fabula) from the approved premise.

## Steps

1. **Structuralist Builds Fabula**
   - Selects Proppian function sequence (or alternative)
   - Defines narrative programs with canonical schema
   - Establishes causality chains
   - Applies modality rules
   - Validates each event through GOLEM model (goal → action → outcome → event → perception → internal element)

2. **Structuralist Checks Constraints**
   - Validates against Greimas action/state logic
   - Validates against causality, modality, no-filler rules
   - Ensures genre compliance
   - Validates Todorov equilibrium arc

3. **Theme Specialist Validates Thematic Fit**
   - Ensures fabula instantiates declared themes
   - Flags thematic drift

4. **Showrunner Reviews Structure**
   - Validates fabula coherence
   - Approves or requests revision

## Output
- Fabula chain with Greimas diagnostics per event
- Story bible (structure section)
- Ready for Workflow 03
