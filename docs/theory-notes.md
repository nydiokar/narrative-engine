# Theory Notes

## Greimasian Semiotics — Core Structural Layer

A.J. Greimas's actantial model is the **primary structural layer** of the engine. It sits above Propp and defines *why* narrative actions must exist, what value-object is being contested, who desires it, who blocks it, who authorizes it, who benefits, and whether each episode actually transforms the fabula.

### The Six Actantial Roles

| Role | Function | Canonical Question |
|------|----------|-------------------|
| **Subject** | Desires / pursues the Object | Who acts? |
| **Object** | Value-state under contest | What value is contested? |
| **Sender** | Authorizes / instigates the quest | Who sets action in motion? |
| **Receiver** | Benefits from Object attainment | Who benefits? |
| **Helper** | Enables Subject (provides modalities) | Who enables? |
| **Opponent** | Blocks Subject (via competing value-logic) | Who hinders? |

### Key Insight: Actants vs Characters

Characters become actants, not fixed "people." One character can shift actantial position depending on the narrative program. This is essential for autonomous AI writing because it prevents flat archetype-following.

### Object of Value = Value-State

The Object is not limited to physical items. It is a **value-state**: love, recognition, freedom, truth, revenge, belonging, legitimacy, inheritance, forgiveness, control, restoration of order, escape from shame.

### Canonical Narrative Schema

Every narrative program follows four phases: **Manipulation → Competence → Performance → Sanction**.

### Action / State Distinction (действие / състояние)

- **действие (action)**: an operation that transforms a state.
- **състояние (state)**: a configuration of actantial junctions, modality values, and Object-of-value relations.

Rule: Every meaningful action must transform a state. Every important state must condition possible actions.

## Proppian Morphology — Sequence Layer

Vladimir Propp identified 32 narrative functions in Russian folktales. The engine uses Propp for the **deep action grammar** — a reusable sequence of functions for folktale, mythic, quest, transformation, rescue, revenge, trial, and return structures.

**Role in the engine**: Propp defines one possible morphology of *how* the Greimas-defined movement unfolds. Propp is valid but limited — it does not specify audience promise, genre positioning, theme hierarchy, value conflict, emotional arc, character modelling, world ontology, or editorial finishing.

## Character Theory: Layered Model

There is no universally accepted exhaustive taxonomy for all character types. The engine separates character into seven layers:

| Layer | Framework |
|-------|-----------|
| Function | Propp / Greimas actantial roles |
| Personality | Five-Factor Model (Big Five) |
| Values | Schwartz Theory of Basic Values |
| Social Mode | Relational Models Theory (4 models) |
| Attachment | Attachment frameworks (4 patterns) |
| Motivation | SDT + Reiss 16 Desires + Goal Polarity |
| Emotion | Plutchik's Wheel of Emotions |

## Emotion Model

Plutchik's eight basic emotion dimensions: Joy, Trust, Fear, Surprise, Sadness, Disgust, Anger, Anticipation — each with three intensity levels and combinatorial blends (e.g., Joy + Trust = Love).

---

## Conflict Typology

Seven types: Internal, Interpersonal, Institutional, Environmental, Epistemic, Metaphysical, Systemic. A story with only one active conflict type risks flatness.

---

## Fabula vs Sujet

- **Fabula**: the raw chronological sequence of events, validated by the Greimas action/state diagnostic.
- **Sujet**: the order and manner in which events are presented (discourse layer).

---

## Todorov's Equilibrium Model

A third structural simplification: **Equilibrium → Disruption → Restored Equilibrium**. Used alongside Greimas and Propp as an additional macro-arc validator.

---

## GOLEM-Inspired Fabula Modelling

Narrative events are modelled as: **goal → action → outcome → event → perception → internal element**. This is closer to what an LLM can store, retrieve, and manipulate than vague archetypes.

---

## Complete Narrative Ontology Summary

The full stack (12 layers): Brief → Concept → Structure → Fabula → Theme → Character → Conflict → World → Discourse → Control → Editorial → Evaluation.
