# Cliché Definition

## Operational Definition

**Cliché** = high-frequency genre defaults assembled without meaningful inversion, escalation, recombination, or thematic necessity.

**Fresh** = a low-similarity or unexpectedly recombined candidate that still passes the hard gate (structural soundness).

## Cliché Signals

Detect cliché when any of these are present:

| Signal | Description |
|--------|-------------|
| **Default genre setting** | The most obvious world for the genre (medieval tavern for fantasy, rain-slicked street for noir) |
| **Default protagonist type** | The expected protagonist archetype for the genre (chosen one in fantasy, grizzled detective in crime) |
| **Default motivation** | The obvious reason for acting (save the world, solve the case) |
| **Default villain motive** | Villain is evil because evil |
| **Default ending** | The genre-standard resolution (wedding for romance, villain defeated for fantasy) |
| **No cost for victory** | The protagonist wins everything, loses nothing |
| **No contradiction inside protagonist** | Internally consistent in a way that feels unreal |
| **Theme stated but not dramatized** | The narrator says what the story is about, but no scene embodies it |
| **World exists only as decoration** | Setting has no impact on choices or outcomes |
| **Villain is evil because evil** | No value-logic behind the opposition |
| **Chosen one without burden** | Destiny without cost |
| **Revenge without moral deformation** | The protagonist exacts revenge and remains unchanged |
| **Romance without specific incompatibility** | Lovers are kept apart by vague circumstances, not genuine value conflict |
| **Mentor dies only to motivate hero** | Death has no other narrative function |

## Freshness Generators

To remediate cliché, apply one or more of these operations:

| Generator | How |
|-----------|-----|
| **Change Object of Value** | Make the protagonist want something unexpected |
| **Change moral cost** | Raise the price of pursuing the Object |
| **Change actant roles** | Swap who occupies which structural position |
| **Invert helper/opponent** | The helper has a hidden cost; the opponent has a valid point |
| **Make world enforce theme** | The setting's rules force thematic choices |
| **Give antagonist a valid value** | The opponent is right about something |
| **Move genre promise into unexpected world** | Romance in a war zone; fantasy in a bureaucracy |
| **Make victory damage something real** | Success has a permanent cost |
| **Make wound contradict desire** | What the character wants conflicts with their deepest damage |
| **Make helper dangerous** | Aid comes with strings |
| **Make love interest ideologically opposed** | Attraction across a value chasm |
| **Make final sanction ambiguous** | The ending unsettles easy judgment |
| **Make protagonist right tactically but wrong morally** | They win the battle but lose themselves |
| **Make antagonist wrong methodologically but right diagnostically** | Their diagnosis is accurate even if their methods are not |

## Freshness Formula

```
Fresh = familiar structure + unexpected value logic + specific cost
```

| Type | Description |
|------|-------------|
| **Bad freshness** | Random twist, arbitrary subversion, genre betrayal, shock without consequence |
| **Good freshness** | A twist that reveals the true Object of Value; a reversal that changes actantial positions; a victory that exposes the protagonist's false belief |

## Relationship to Evaluation

Cliché detection feeds into the **Novelty** dimension of the soft gate. A candidate is not rejected for cliché alone — but its novelty score is penalised proportionally to the number and severity of unmitigated clichés.
