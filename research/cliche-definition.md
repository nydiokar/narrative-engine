# Cliché Definition

## Operational Definition

**Cliché** = high-frequency genre defaults assembled without meaningful inversion, escalation, recombination, or thematic necessity.

**Fresh** = a low-similarity or unexpectedly recombined candidate that still passes the hard gate (structural soundness).

## Detection Heuristics

A narrative element is flagged as potentially cliché when:

1. **High-frequency pattern**: the element matches a known high-frequency pattern for its genre (e.g., "chosen one" in epic fantasy, "amnesia" in mystery).

2. **No inversion**: the pattern is deployed straight — no expectation subversion, no role reversal, no power dynamic shift.

3. **No escalation**: the pattern is not pushed beyond its typical scope or stakes.

4. **No recombination**: the element is used in isolation rather than hybridised with an unexpected second pattern.

5. **No thematic necessity**: the element's presence cannot be justified by the story's declared themes — it is there because "that's how this genre works."

## Mitigation Strategies

When a cliché is detected:

| Strategy | Description |
|----------|-------------|
| **Invert** | Flip the expected power dynamic or outcome. |
| **Escalate** | Raise the stakes beyond the typical ceiling. |
| **Recombine** | Cross the pattern with an unexpected genre or theme. |
| **Ground in theme** | Rewrite so the element flows from thematic necessity, not convention. |
| **Substitute** | Replace with a structurally equivalent but less common pattern. |

## Relationship to Evaluation

Cliché detection feeds into the **Novelty** dimension of the soft gate. A candidate is not rejected for cliché alone — but its novelty score is penalised proportionally to the number and severity of unmitigated clichés.
