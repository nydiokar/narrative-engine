# Worldbuilding Dimensions

There is no universally accepted exhaustive ontology for world types. The engine represents worldbuilding dimensionally rather than attempting a final list.

## Core Dimensions

| Dimension | Description | Example Questions |
|-----------|-------------|-------------------|
| **Physical Environment** | Geography, climate, ecology, natural resources | What is the terrain? What is the climate? What flora/fauna exist? |
| **Institutions** | Government, legal systems, education, religion | Who holds power? How is justice administered? |
| **Technology / Magic** | Technical capability, magical systems, energy sources | What is possible? What are the costs and limits? |
| **Historical Depth** | Past events, founding myths, remembered history | What happened before the story starts? What is forgotten? |
| **Objects & Resources** | Common objects, rare resources, economic foundations | What do people use, trade, and value? |
| **Sensory / Aesthetic** | Architecture, art, fashion, food, daily sensory texture | What does the world look, sound, smell, taste like? |
| **Interaction Rules** | Causality, physics, social norms, taboos | What is the cause-effect structure? What is forbidden? |
| **Receiver Experience** | How the world is revealed to the audience | Is the world explained or absorbed? What is familiar vs strange? |

## World Ontology Approach

Worldbuilding is modelled as a set of axis sliders rather than categorical types:

| Axis | Range |
|------|-------|
| Realism | Historical → Speculative → Fantastic |
| Scope | Single room → City → Continent → World → Multiverse |
| Familiarity | Contemporary → Historical → Alternate → Alien |
| Rule Explicitness | Implicit → Partial → Fully defined |
| Magical Density | None → Low → Medium → High → Pervasive |
| Technological Level | Prehistoric → Medieval → Industrial → Information → Post-human |
| Social Complexity | Homogeneous → Stratified → Complex → Fractured |

## Role in the System

Worldbuilding dimensions feed into:
- World contract (setting rules and constraints)
- Worldbuilder agent brief
- Continuity Editor consistency checks
- Critic evaluation (do world rules hold across the fabula?)
- Scene Writer atmospheric rendering
