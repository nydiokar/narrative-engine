# Sequence Types

A sequence is a connected series of scenes that forms a coherent narrative unit within an episode. Sequence types provide reusable macro-patterns for scene grouping.

## Sequence Types

| Type | Core Pattern | Typical Scene Types Included | Typical Episode Phase |
|------|-------------|------------------------------|------------------------|
| **Quest** | Seek → Test → Obtain | Decision, Discovery, Test, Confrontation | Performance |
| **Investigation** | Clue → Inference → Revelation | Discovery, Interrogation, Revelation | Competence / Performance |
| **Training** | Deficiency → Instruction → Qualification | Introduction, Training, Test | Competence |
| **Pursuit** | Chase → Trap → Escape | Chase, Confrontation, Escape | Performance |
| **Infiltration** | Approach → Breach → Extraction | Decision, Confrontation, Escape | Performance |
| **Courtship** | Attraction → Obstacle → Union/Separation | Introduction, Seduction, Decision, Confrontation | Manipulation / Performance |
| **Trial** | Accusation → Defense → Judgment | Interrogation, Confrontation, Judgment | Performance / Sanction |
| **Battle** | Skirmish → Climax → Aftermath | Confrontation, Battle, Quiet Aftermath | Performance |
| **Escape** | Capture → Struggle → Flight | Confrontation, Escape, Chase | Performance |
| **Descent** | Comfort → Crisis → Fall | Decision, Dark Night, Reversal | Performance |
| **Revelation** | Concealment → Discovery → Exposure | Discovery, Revelation, Recognition | Sanction |
| **Transformation** | Old Self → Crisis → New Self | Decision, Dark Night, Resolution | Performance / Sanction |
| **Betrayal** | Trust → Deception → Exposure | Introduction, Betrayal, Confrontation | Performance |
| **Reconciliation** | Rupture → Approach → Union | Confrontation, Negotiation, Resolution | Sanction |
| **Corruption** | Innocence → Temptation → Fall | Temptation, Decision, Reversal | Performance |
| **Redemption** | Guilt → Atonement → Restoration | Decision, Sacrifice, Recognition | Performance / Sanction |
| **Return** | Departure → Journey → Arrival | Decision, Discovery, Resolution | Performance / Sanction |

## Sequence Selection

- A sequence type must serve the episode's dominant canonical phase.
- Sequence types have default beginning/middle/end scene type templates.
- Sequences can nest: a Quest sequence may contain an Infiltration sequence.
- The sequence type determines the expected state trajectory (opening → closing).
