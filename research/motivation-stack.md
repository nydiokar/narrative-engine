# Motivation Stack

Character motivation is modelled as a stacked system of four layers, each answering a different "why".

## Stack Layers

| Layer | Framework | Question Answered |
|-------|-----------|-------------------|
| 1. Basic Needs | Self-Determination Theory | What does the character require to function? |
| 2. Basic Desires | Reiss 16-Desire Model | What does the character innately seek? |
| 3. Values | Schwartz Values | What does the character see as worth pursuing? |
| 4. Goal Polarity | Fabula models | What is the character's current directional stance? |

## Layer 1: Basic Psychological Needs (SDT)

- **Autonomy** — need to feel volitional and self-endorsing
- **Competence** — need to feel effective and capable
- **Relatedness** — need to feel connected to others

Frustration of these needs drives psychological conflict (internal conflict type).

## Layer 2: Reiss 16 Basic Desires

1. Power — desire for influence
2. Curiosity — desire for knowledge
3. Independence — desire for self-reliance
4. Social Contact — desire for peer companionship
5. Status — desire for social standing
6. Vengeance — desire to get even
7. Honour — desire for loyalty and character
8. Idealism — desire for social justice
9. Order — desire for organisation
10. Saving — desire to collect
11. Acceptance — desire for approval
12. Family — desire to raise children
13. Romance — desire for sex and beauty
14. Eating — desire for food
15. Physical Activity — desire for exercise
16. Tranquillity — desire for emotional calm

Each desire has an intensity score (1-10) per character.

## Layer 3: Schwartz Values

(See `/research/character-layers.md` for the full list.)

Values operate at a higher abstraction than desires — they are the character's conscious principles, not innate drives.

## Layer 4: Goal Polarity

Every narrative action is driven by a goal polarity:

| Polarity | Meaning |
|----------|---------|
| **Attain** | The character seeks to acquire or reach something |
| **Maintain** | The character seeks to preserve a current state |
| **Leave** | The character seeks to escape or abandon something |
| **Avoid** | The character seeks to prevent something from happening |

## Operational Use

The motivation stack feeds into:
- Character contract modality fields (wanting modality is now grounded in specific desires/values)
- Scene-level action justification (why does the character perform this action?)
- Conflict design (conflicting desires or values between characters)
- Critic evaluation (is character action motivationally coherent?)
